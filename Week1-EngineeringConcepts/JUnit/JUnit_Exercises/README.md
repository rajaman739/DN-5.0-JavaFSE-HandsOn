## Exercise 4 - AAA Pattern, Setup and Teardown

### Arrange
Prepare the objects and input values.

### Act
Call the method being tested.

### Assert
Verify the expected output using assertions.

### Setup Method

`@Before` initializes the required objects before each test.

### Teardown Method

`@After` cleans up resources after each test.
