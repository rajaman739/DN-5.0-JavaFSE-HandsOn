# Exercise 3 - Stored Procedures

## Scenario 1
Created a procedure `ProcessMonthlyInterest` to apply 1% interest to all savings accounts.

## Scenario 2
Created a procedure `UpdateEmployeeBonus` that updates employee salaries using a department name and bonus percentage passed as parameters.

## Scenario 3
Created a procedure `TransferFunds` that transfers money between two accounts after checking if the source account has sufficient balance.
